# Redesign My Landing Page

Build, critique, and iterate high-converting landing pages using React, Vite, TypeScript, Tailwind, shadcn/ui, and Iconify.

## Install

```bash
# Codex
npx skills add instructa/agent-skills --skill redesign-my-landingpage --agent codex

# Claude Code
npx skills add instructa/agent-skills --skill redesign-my-landingpage --agent claude-code

# Cursor
npx skills add instructa/agent-skills --skill redesign-my-landingpage --agent cursor
```

## Use When

- The user asks for a landing page, sales page, signup page, or CRO improvement.
- An existing page needs a conversion, copy, proof, CTA, design, or accessibility audit.
- The deliverable should include production-ready shadcn + Vite code.

## Default Stack

- React + Vite + TypeScript
- Tailwind CSS
- shadcn/ui components
- Iconify icons through `@iconify/react`
